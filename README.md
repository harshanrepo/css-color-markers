🎨 CSS Color Markers

A simple and creative CSS design project that recreates colored markers using HTML and CSS. This project focuses on gradients, shadows, and basic layout styling.

---

🚀 Live Demo

👉 https://harshanrepo.github.io/css-color-markers/

---

## 📌 Features

* Colored markers using CSS gradients
* Realistic shadow effects with box-shadow
* Clean and centered layout
* Simple and readable structure

---

🛠️ Built With

* HTML5
* CSS3 (gradients, box-shadow)

---

📚 What I Learned

* Using linear gradients in CSS
* Applying box-shadow for visual depth
* Structuring simple UI components

---

## 🔧 How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
